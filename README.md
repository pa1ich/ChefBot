## 🚀 Что умеет бот

| Функция | Как работает |
|---------|-------------|
| 🎤 Голосовой ввод | Распознавание через **Whisper** (локально) |
| 🔍 Поиск по книгам | Векторный поиск в **Pinecone** (6 кулинарных книг) |
| 🤖 Генерация ответа | Локальная LLM **Qwen2.5:3b** через Ollama |
| 📚 Указание источника | Формат: `[Книга: название, стр. X]` |
| 🔐 Авторизация | Команда `/auth <пароль>` для доступа |

## 🧱 Архитектура
```yaml
Бот:
  • aiogram 3.x — асинхронный Telegram-бот
  • Python 3.10+

ASR (распознавание речи):
  • openai-whisper — локальная транскрипция
  • librosa + soundfile — обработка аудио

RAG-пайплайн:
  • pinecone-client — векторная база данных
  • sentence-transformers: multilingual-e5-large — эмбеддинги
  • pypdf + langchain — парсинг и чанкинг текстов

LLM:
  • Ollama + qwen2.5:3b — локальная генерация ответов
  • Промпт-инжиниринг с контекстом из книг

Инфраструктура:
  • python-dotenv — управление секретами
  • asyncio — асинхронная обработка запросов
